# SaiAerobics Dashboard

A full-stack gym management and fitness tracking platform built for Sai Aerobics.

## Features
- Member login & dashboard
- Daily weight tracking with graphs
- Membership plans & expiry tracking
- AI-powered nutritionist
- Streak & consistency system
- Admin panel for gym instructor

## Tech Stack
Frontend: React, Tailwind CSS  
Backend: Node.js, Express  
Database: MongoDB  
AI: Gemini / OpenAI API
